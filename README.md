# openclaw_windows
windows打包版本，内置环境，安装即用

setup-env.bat是设置环境变量的脚本，如果控制台openclaw --version识别不到版本，可以运行脚本设置环境变量

设置环境变量后，可以同官网版本一样启动（控制台 openclaw gateway），也可以用exe快捷方式启动

定期会同步官网版本，由于官网更新频率过高，各种兼容问题，会等版本稳定后，略晚于官方版本发布

尽量不要安装到C盘目录，部分用户没有读写权限，安装卡住，安装到D盘无限制

目前最新版本基于官方 2026.3.24 版本封装

右边releases下载，分win10+ 和win10-两种版本
